## üëæ Game Concept

* **Genre:** Horror
* **Visual Style:** Low-poly, pixelated PS1 aesthetic.
* **Theme:** 1900s Doom-style undead.
* **Primary Weapon:** Finger Gun

### Enemies
* **Seeker:** A blind, sound-based enemy (similar to "Clickers").
* **Attacker:** A standard zombie with difficulty variants (indicated by eye color).
* **Stationary:** An enemy that only moves when not being looked at (similar to "Weeping Angels").
* **Final Boss:** Necromancer (TBD).

## üéÆ Core Mechanics

* **Pickups:**
    * **Health:** Food
    * **Stamina:** Flasks
    * **Over-shield:** Flasks
    * **Ammo:** "Fingers"
    * **Light Source:** Glow Sticks
    * **Grenade:** "Thumb"
    * **Keys:** Blood Offering / Soul / TBD
* **Environment:**
    * **Elevators:** Used for level/room transference.
    * **Capillaries:** Crawlspaces leading to secret areas.
    * **Enemy Spawning:** Dynamic enemy spawning system.

### Core Player Systems
* First-person controller
* Pickup system
* "Finger gun" shooting system
* "Thumb" grenade

### Basic Enemy AI
* **Seeker:** Wanders, moves toward sound.
* **Attacker:** Vision-based chase AI with difficulty variants (eye color/speed).
* **Stationary:** Freezes completely when in the player's direct view, moves when not.
*(Note: Functional logic is the priority; animations will not be polished.)*

### Early Game Loop & Map
* Greybox/Blockout for the main playable area.
* Includes: First safe room, hallways, 2-3 enemy rooms, 1 capillary secret.
* Functional basic key-door system.
* Elevator system transitions between floors.

### Final Boss (Basic)
* Necromancer AI (summons enemies, potential phases/weak points).

### UI/UX (Basic)
* General map placeholder.
* Item holding area/inventory concept.
* Game intro sequence.
